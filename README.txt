How to run the game:

In the terminal at the directory of the project, run the command 'cabal run' in order to open the game. 

Instructions:

Press 'W' in order to move forward
Press 'A' to turn left or press 'D' to turn right
Press 'SPACE' to shoot the enemies
Press 'P' to pause/unpause the game

Killing enemies will increase the players score
Aim for a highscore!
A new UFO enemy will be able to spawn after 30 seconds of gameplay

Buttons can be pressed through mouse input in the menus

Note: If you want to enter debug mode, 
set the Boolean in src/Constants.hs - Line 10 to True in order to see the debug
